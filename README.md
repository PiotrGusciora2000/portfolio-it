# Dell Inspiron N Series — Kompleksowy serwis układu chłodzenia i modernizacja SSD

## Skrócony opis projektu

Kompleksowy serwis laptopa Dell Inspiron N Series obejmujący pełny demontaż, czyszczenie układu chłodzenia, wymianę pasty termoprzewodzącej oraz modernizację HDD → SSD. Zdiagnozowano i usunięto problem throttlingu oraz wyłączeń termicznych spowodowanych degradacją układu chłodzenia po wieloletniej eksploatacji. Po ponownym montażu przeprowadzono diagnostykę błędu „black screen”, identyfikując niepodłączoną taśmę matrycy LCD. Laptop pozostaje sprawny po ok. 7 latach od wykonania naprawy.

# Problem / objawy

Laptop Dell Inspiron N Series z dedykowanym GPU NVIDIA GeForce GT 525M wykazywał:

- silne przegrzewanie pod obciążeniem
- throttling CPU/GPU podczas gier
- bardzo wysoką temperaturę obudowy i klawiatury
- awaryjne wyłączanie po ok. 2 godzinach pracy pod obciążeniem

Dodatkowo:

- układ chłodzenia był silnie zanieczyszczony kurzem
- pasta termoprzewodząca była wyschnięta po ok. 8–10 latach eksploatacji
- po pierwszym montażu laptop uruchamiał wentylatory i napęd DVD, ale nie wyświetlał obrazu

---

# Diagnostyka / analiza

Zidentyfikowano objawy typowe dla problemów termicznych układów elektronicznych:

- throttling wydajności CPU/GPU
- wysokie temperatury obudowy
- zabezpieczające wyłączanie systemu

Wnioski diagnostyczne:

- degradacja przewodnictwa cieplnego przez starą pastę termoprzewodzącą
- ograniczony przepływ powietrza przez zabrudzony układ chłodzenia

Po wystąpieniu problemu „black screen” po montażu:

- przeprowadzono ponowny demontaż urządzenia
- przeanalizowano dokumentację fotograficzną z rozbiórki
- zidentyfikowano niepodłączoną taśmę matrycy LCD

# Wykonane działania

- Kompletny demontaż laptopa wraz z wyjęciem płyty głównej
- Modernizacja urządzenia poprzez wymianę HDD → SSD
- Demontaż układu chłodzenia CPU/GPU
- Usunięcie starej pasty termoprzewodzącej
- Czyszczenie powierzchni CPU oraz GPU alkoholem izopropylowym (IPA)
- Aplikacja nowej pasty termoprzewodzącej
- Czyszczenie radiatorów i wentylatorów z kurzu
- Ponowny montaż laptopa
- Dokumentowanie etapów rozbiórki zdjęciami referencyjnymi
- Diagnostyka błędu „black screen” po montażu
- Ponowne podłączenie taśmy matrycy LCD

## Dodatkowe działania systemowe

- Utworzenie bootowalnego pendrive instalacyjnego Windows 10
- Instalacja systemu Windows 10 po zakończeniu serwisu sprzętowego
- Konfiguracja i optymalizacja systemu operacyjnego
- Wyłączenie zbędnych usług systemowych (m.in. telemetria, Delivery Optimization)
- Modyfikacja ustawień systemowych oraz wpisów rejestru Windows
- Optymalizacja responsywności i ograniczenie zbędnych procesów działających w tle

# Rozwiązanie / efekt

- Przywrócono pełną sprawność laptopa
- Wyeliminowano problem przegrzewania i wyłączeń termicznych
- Poprawiono kulturę pracy oraz stabilność pod obciążeniem
- Laptop pozostaje sprawny po ok. 7 latach od wykonania serwisu
- Skutecznie rozwiązano problem braku obrazu po montażu bez uszkodzenia komponentów

---

# Wnioski techniczne

- Długotrwała eksploatacja laptopa bez konserwacji prowadzi do degradacji układu chłodzenia i throttlingu
- Konstrukcja laptopa wymagała pełnego demontażu płyty głównej nawet do wymiany dysku
- Dokumentacja fotograficzna znacząco redukuje ryzyko błędów podczas ponownego montażu
- Brak obrazu po serwisie może wynikać z prostych błędów montażowych mimo poprawnej pracy pozostałych podzespołów
